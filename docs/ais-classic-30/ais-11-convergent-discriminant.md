# AIS經典論文 11：多特質多方法矩陣的收斂與區辨效度

**English Title**: Convergent and discriminant validation by the multitrait-multimethod matrix
**中文標題**: 多特質多方法矩陣的收斂與區辨效度
**作者**: Campbell, D. T., & Fiske, D. W.
**年份**: 1959
**期刊**: Psychological Bulletin
**卷期**: 56(2), 81–105
**DOI**: [10.1037/h0046016](https://doi.org/10.1037/h0046016)

## 📌 第一張：核心觀點卡

**本論文提出多特質多方法矩陣（MTMM），主張有效的構念測量必須同時具備收斂效度（同一特質用不同方法測量應高度相關）和區辨效度（不同特質即使用相同方法測量也應低度相關），挑戰「單一測量工具足以驗證構念」的假設**

### 收斂效度 vs 區辨效度：雙重驗證標準

| 效度類型 | 英文 | 定義 | 驗證方法 | 標準 | IS 範例 |
|---------|------|------|---------|------|---------|
| **收斂效度** | Convergent Validity | **同一構念**用**不同方法**測量，結果應**高度相關** | 不同測量方法的相關係數 | r &gt; 0.7（同構念異方法） | 問卷法 + 行為觀察法測「使用意圖」應高度一致 |
| **區辨效度** | Discriminant Validity | **不同構念**即使用**相同方法**測量，結果應**低度相關** | 不同構念間的相關係數 | r &lt; 0.5（異構念同方法） | 「知覺有用性」vs「知覺易用性」問卷分數相關應低 |

**核心邏輯**：
- **收斂效度**：確保測量工具真的測到該構念（信號強度）
- **區辨效度**：確保測量工具不會測到其他構念（雜訊控制）
- **雙重標準**：兩者缺一不可，單有收斂效度可能測到共同方法偏差（method bias）

---

### MTMM 矩陣結構與判讀規則

**多特質多方法矩陣**：交叉設計矩陣，同時檢驗多個特質（構念）與多個方法（測量工具）的組合

#### MTMM 範例：3 特質 × 2 方法（簡化版）

| | 方法 1：問卷 | | | 方法 2：實驗 | | |
|---|---|---|---|---|---|---|
| | T1 | T2 | T3 | T1 | T2 | T3 |
| **方法 1：問卷** | | | | | | |
| T1（知覺有用性） | **(1.00)** | 0.35 | 0.28 | **0.72** | 0.22 | 0.18 |
| T2（知覺易用性） | 0.35 | **(1.00)** | 0.40 | 0.30 | **0.68** | 0.25 |
| T3（使用意圖） | 0.28 | 0.40 | **(1.00)** | 0.25 | 0.32 | **0.75** |
| **方法 2：實驗** | | | | | | |
| T1（知覺有用性） | **0.72** | 0.30 | 0.25 | **(1.00)** | 0.38 | 0.30 |
| T2（知覺易用性） | 0.22 | **0.68** | 0.32 | 0.38 | **(1.00)** | 0.45 |
| T3（使用意圖） | 0.18 | 0.25 | **0.75** | 0.30 | 0.45 | **(1.00)** |

**圖例說明**：
- **對角線（1.00）**：信度係數（同一測量的重測相關）
- **粗體數字（0.72, 0.68, 0.75）**：**效度對角線**（同一特質不同方法）→ 收斂效度證據
- **斜體數字（0.35, 0.40...）**：異特質同方法三角區 → 區辨效度證據
- **普通數字（0.22, 0.18...）**：異特質異方法三角區 → 最低相關

#### Campbell-Fiske 判讀四大準則

1. **準則 1（收斂效度）**：效度對角線數值應**顯著異於零且夠大**
   - 標準：r > 0.5（理想 > 0.7）
   - 範例：T1 問卷 vs T1 實驗 = 0.72 ✓

2. **準則 2（區辨效度 - 異方法比較）**：效度對角線應**高於**同列同欄的異特質異方法數值
   - 標準：0.72（T1 效度）> 0.22（T2 異特質異方法）✓
   - 意義：同構念異方法相關 > 異構念異方法相關

3. **準則 3（區辨效度 - 同方法比較）**：效度對角線應**高於**異特質同方法數值
   - 標準：0.72（T1 效度）> 0.35（T1-T2 同方法）✓
   - 意義：同構念異方法相關 > 異構念同方法相關

4. **準則 4（方法偏差檢驗）**：異特質同方法三角區的相關模式應**類似於**異特質異方法三角區
   - 檢查：若 T1-T2 同方法相關（0.35）遠大於 T1-T2 異方法相關（0.22），表示方法偏差過大 ⚠️
   - 理想：兩個三角區的構念間相關排序應一致

### 醫療 AI 系統構念驗證範例

**情境**：驗證「AI 診斷信任度」構念的收斂與區辨效度

#### 研究設計

**特質（構念）**：
- T1：AI 診斷信任度
- T2：AI 系統易用性
- T3：醫師專業自信

**方法（測量工具）**：
- M1：自陳式問卷（Likert 量表）
- M2：情境模擬實驗（實際使用 AI 後評分）
- M3：行為觀察（實際採納 AI 建議的頻率）

#### MTMM 矩陣（簡化版）

| | M1 問卷 | | | M2 實驗 | | |
|---|---|---|---|---|---|---|
| | T1 | T2 | T3 | T1 | T2 | T3 |
| **M1 問卷** | | | | | | |
| T1（AI 信任） | (1.00) | **0.45** | **0.32** | **0.78** | 0.28 | 0.20 |
| T2（易用性） | **0.45** | (1.00) | 0.25 | 0.30 | **0.72** | 0.18 |
| T3（專業自信） | **0.32** | 0.25 | (1.00) | 0.22 | 0.20 | **0.68** |
| **M2 實驗** | | | | | | |
| T1（AI 信任） | **0.78** | 0.30 | 0.22 | (1.00) | **0.42** | **0.30** |
| T2（易用性） | 0.28 | **0.72** | 0.20 | **0.42** | (1.00) | 0.22 |
| T3（專業自信） | 0.20 | 0.18 | **0.68** | **0.30** | 0.22 | (1.00) |

**效度驗證結論**：

✅ **收斂效度成立**：
- AI 信任（M1 vs M2）= 0.78 > 0.7 ✓
- 易用性（M1 vs M2）= 0.72 > 0.7 ✓
- 專業自信（M1 vs M2）= 0.68 ≈ 0.7 △

⚠️ **區辨效度部分成立**：
- AI 信任 vs 易用性（同方法 M1）= 0.45，但異方法僅 0.28
  - 問題：問卷法可能混淆「信任」與「易用性」
- AI 信任 vs 專業自信（同方法 M1）= 0.32，但異方法僅 0.20
  - 解讀：兩構念確實有區辨，但問卷法略有共同方法偏差

**改進建議**：
1. 重新設計問卷題目，降低 AI 信任與易用性的語意重疊
2. 增加第三種方法（如專家評分）進一步驗證
3. 考慮使用驗證性因素分析（CFA）補充 MTMM 證據

### 為什麼需要「多方法」？單一量表的致命缺陷

**問題情境**：只用問卷測量「知覺有用性」，信度係數 α = 0.92（很高）

❌ **錯誤結論**：「我的測量工具很好！」

⚠️ **隱藏風險**：
1. **共同方法變異（Common Method Variance, CMV）**：
   - 問卷法本身的特性（如社會期許、情緒狀態）虛增相關
   - 實例：受試者心情好時，所有問卷題目都打高分

2. **構念混淆（Construct Confusion）**：
   - 可能同時測到「知覺有用性」+ 「知覺易用性」
   - 單一方法無法偵測這種混淆

3. **方法特定變異（Method-Specific Variance）**：
   - 測到的可能是「填問卷的能力」而非「知覺有用性」
   - 實例：語文能力強的人在所有量表都得高分

**MTMM 如何破解**：
- **加入行為觀察法**：測量實際使用時間（客觀指標）
- **比較兩種方法結果**：
  - 若問卷 vs 行為相關低（< 0.5）→ 問卷可能測到其他東西
  - 若問卷 vs 行為相關高（> 0.7）→ 兩種方法都測到同一構念 ✓

### 心理計量學三大效度的關係

| 效度類型 | 回答問題 | 驗證方法 | MTMM 對應 |
|---------|---------|---------|----------|
| **內容效度** (Content Validity) | 測量內容是否涵蓋構念定義範圍？ | 專家判斷、文獻回顧 | （前置條件）確保特質定義清晰 |
| **準則效度** (Criterion Validity) | 測量結果是否能預測外在準則？ | 與黃金標準相關 | 可用外在準則作為 MTMM 中的一個「方法」 |
| **構念效度** (Construct Validity) | 測量是否真的測到理論構念？ | **MTMM 矩陣** | ✓ 收斂效度 + 區辨效度 |

**重點**：Campbell-Fiske 將構念效度拆成「收斂」與「區辨」兩個可操作化的標準，使效度驗證從「專家主觀判斷」提升為「量化矩陣檢驗」

### 📊 MTMM 在 IS 研究的三大應用情境

#### 應用 1：量表開發驗證（最常見）

**情境**：開發新的「AI 接受度量表」

**MTMM 設計**：
- **特質**：AI 有用性、AI 易用性、使用意圖
- **方法**：問卷、實驗任務完成時間、訪談評分

**驗證重點**：
1. 確認三個構念真的獨立（區辨效度）
2. 確認問卷測到的與實驗測到的一致（收斂效度）

---

#### 應用 2：跨文化量表適用性檢驗

**情境**：TAM 量表是否適用於台灣醫療情境？

**MTMM 設計**：
- **特質**：知覺有用性（美國版定義）、知覺有用性（台灣版定義）
- **方法**：原始英文問卷、翻譯中文問卷、本土化重新設計問卷

**檢驗邏輯**：
- 若「美國版」vs「台灣版」效度對角線低 → 構念意義改變，需重新定義
- 若「翻譯版」vs「原始版」效度對角線低 → 翻譯問題

---

#### 應用 3：共同方法偏差診斷

**情境**：問卷研究發現「知覺有用性 → 使用意圖」相關過高（r = 0.95），懷疑 CMV

**MTMM 設計**：
- **特質**：知覺有用性、使用意圖
- **方法**：問卷（同一時間點）、問卷（時間分離）、客觀使用紀錄

**診斷方法**：
- 比較「同方法異特質」vs「異方法異特質」相關
- 若同方法相關（0.95）遠大於異方法相關（0.60）→ 嚴重 CMV
- 解決：程序補救（分時測量）、統計補救（控制潛在方法因子）

## ✍️ 第二張:改寫卡

### 為什麼這篇論文是方法學革命？

**1959 年前的效度驗證困境**：

❌ **當時主流做法**：
- 內容效度：找幾位專家看題目「感覺對不對」→ 主觀
- 準則效度：與已知量表相關 → 循環論證（如何確定已知量表有效？）
- 因素分析：證明題目歸為同一因素 → 只驗證內部一致性，無法排除方法偏差

⚠️ **致命問題**：研究者可能用單一問卷測量多個構念，發現相關很高，就宣稱「構念有效」
- 實際上：高相關可能來自「共同方法」而非「真實構念關係」

---

**Campbell-Fiske 的破解方法**：

✅ **MTMM 創新**：
1. **雙重驗證標準**：不只要求「同構念同方法」相關高，還要求「同構念異方法」也高
2. **明確化方法效應**：強迫研究者面對「方法變異」vs「特質變異」的區分
3. **矩陣化思維**：將抽象的效度概念轉化為可檢驗的數值模式

**舉例說明**：

| 情境 | 傳統效度驗證 | MTMM 驗證 | 結論差異 |
|------|-------------|----------|---------|
| 問卷測「知覺有用性」，內部一致性 α = 0.95 | ✓ 效度良好 | 加入行為觀察法，問卷 vs 行為相關僅 0.35 | ✗ 問卷可能測到「填答偏好」而非「有用性」 |
| 問卷測「有用性」vs「易用性」，相關 0.85 | ✓ 兩構念高度相關 | MTMM 發現同方法異構念（0.85）&gt; 異方法同構念（0.50） | ✗ 高相關來自方法偏差，非真實構念關係 |

**革命性影響**：
- **方法學典範轉移**：從「單一測量工具驗證」→「多元方法交叉驗證」
- **嚴謹性提升**：迫使研究者設計多種測量方法，增加研究成本但提高效度信心
- **後續發展**：啟發結構方程模型（SEM）中的多特質多方法 CFA 分析

### Campbell-Fiske 矩陣的理論基礎：操作主義哲學

**操作主義（Operationalism）核心主張**：
> 科學構念的意義由其測量操作定義，但單一操作定義不足以完全捕捉構念

**MTMM 的哲學回應**：
1. **構念 ≠ 測量**：「智力」不等於「IQ 分數」，IQ 只是智力的一種測量
2. **多元操作定義**：用多種方法逼近構念真實意義
   - 問卷測「態度」→ 捕捉認知評價
   - 實驗測「態度」→ 捕捉行為傾向
   - 兩者交集 → 更接近「態度」本質

**IS 領域應用**：
- 「系統品質」可由客觀指標（回應時間）、主觀評分（問卷）、專家稽核（第三方評估）多元測量
- MTMM 確保這些測量都指向同一構念，而非各自測到不同東西

---

### MTMM 的數學本質：分解總變異

**測量分數的變異來源**：

```
觀察分數變異 = 特質變異 + 方法變異 + 誤差變異
Total Variance = Trait Variance + Method Variance + Error Variance
```

**MTMM 矩陣的分解邏輯**：

| 相關類型 | 數學意義 | 包含變異 | 理想值 |
|---------|---------|---------|--------|
| **同特質同方法**（信度） | 重測相關 | 特質 + 方法 + 誤差 | 接近 1.00 |
| **同特質異方法**（效度對角線） | 收斂效度 | **特質** | &gt; 0.7 |
| **異特質同方法** | 方法偏差 | 方法（+ 些微特質） | &lt; 0.5 |
| **異特質異方法** | 基線相關 | 誤差（幾乎為 0） | 接近 0 |

**關鍵推論**：
- 若「同特質異方法」> 「異特質同方法」→ **特質變異 > 方法變異** ✓
- 若「同特質異方法」< 「異特質同方法」→ **方法變異 > 特質變異** ✗（嚴重方法偏差）

**實例計算**：

假設觀察到：
- 同特質異方法相關 = 0.72
- 異特質同方法相關 = 0.45

推論：
- 特質變異佔比 ≈ 72%（0.72²）
- 方法變異佔比 ≈ 45%（0.45²）但其中包含誤差
- 結論：特質訊號強於方法雜訊，測量有效 ✓

### IS 領域的 MTMM 經典應用案例

#### 案例 1：Straub (1989) - 電子郵件接受度研究

**研究問題**：驗證「知覺有用性」構念在電子郵件情境的效度

**MTMM 設計**：
- **特質**：知覺有用性、知覺易用性
- **方法**：問卷、實際使用時間記錄、主管評分

**結果**：
- 問卷 vs 使用記錄（同特質異方法）= 0.68 ✓
- 有用性 vs 易用性（同方法異特質）= 0.42 ✓
- **結論**：知覺有用性確實是獨立構念，問卷測量有效

**影響**：成為 TAM 量表效度驗證的範本

#### 案例 2：Gefen & Straub (2005) - 電子商務信任量表

**研究問題**：「信任」vs「風險知覺」是否為獨立構念？

**MTMM 設計**：
- **特質**：信任、風險知覺、購買意圖
- **方法**：問卷、實驗操弄後評分、實際交易行為

**發現**：
- 信任 vs 風險（同方法）相關 = -0.72（負相關很高）
- 但異方法同特質相關也達 0.65-0.70
- **爭議**：兩構念高度相關但仍具區辨性

**後續研究**：引發「信任 vs 風險是否為同一構念兩端」的理論辯論

#### 案例 3：Podsakoff et al. (2003) - CMV 偵測與補救

**研究問題**：問卷研究中的共同方法偏差有多嚴重?

**MTMM 後設分析**：
- 分析 300+ 篇 IS 論文
- 發現平均「同方法異特質」相關 = 0.52（過高！）
- 平均「異方法同特質」相關 = 0.58（應更高！）

**結論**：
- IS 領域普遍存在 CMV 問題
- 建議：程序補救（分時測量、匿名設計）+ 統計補救（控制方法因子）

**影響**：成為 IS 研究方法檢核的必讀文獻

## ❓ 第三張：Q&A 卡

### Q1：MTMM 四大判讀準則為什麼這樣設計？各自回應什麼效度問題？

**準則 1：效度對角線顯著異於零且夠大（r > 0.5）**

📌 **回應問題**：收斂效度 - 不同方法測量同一構念是否一致？

**邏輯**：
- 若同構念異方法相關很低 → 兩種方法可能測到不同東西
- 範例：問卷測「系統滿意度」= 7 分，但實驗觀察使用行為顯示常抱怨
  - 效度對角線 = 0.25（很低）→ 問卷可能測到「社會期許」而非真實滿意度

**統計補充**：
- 理想值 > 0.7（中大效果量）
- 最低門檻 > 0.5（中等效果量）
- 若 < 0.3 → 嚴重質疑測量效度

---

**準則 2：效度對角線 > 同列同欄的異特質異方法相關**

📌 **回應問題**：區辨效度 - 不同構念即使用不同方法測量，相關也應更低

**邏輯**：
- 同構念異方法相關（0.72）應大於異構念異方法相關（0.28）
- 若兩者接近 → 測量工具無法區分不同構念

**範例矩陣檢查**：

| | 方法 1 | 方法 2 |
|---|---|---|
| T1（有用性） | - | **0.72** ← 效度對角線 |
| T2（易用性） | - | 0.28 ← 異特質異方法 |

判斷：0.72 > 0.28 ✓（符合準則 2）

**違反範例**：
- 若異特質異方法相關 = 0.68（接近 0.72）
- 意義：T1 與 T2 可能不是獨立構念，或測量工具混淆兩者

---

**準則 3：效度對角線 > 異特質同方法相關**

📌 **回應問題**：方法偏差控制 - 同方法測量不同構念的相關應低於異方法測量同構念

**邏輯**：
- 若異特質同方法相關（0.65）> 同特質異方法相關（0.60）
- 代表「方法效應」> 「特質效應」→ 方法偏差過大

**範例矩陣檢查**：

| | M1-T1 | M1-T2 | M2-T1 |
|---|---|---|---|
| M1-T1（問卷有用性） | 1.00 | **0.65** ← 異特質同方法 | **0.60** ← 效度對角線 |

判斷：0.60 < 0.65 ✗（違反準則 3）
- 問題：問卷法的方法效應過強，導致所有問卷題目都高度相關
- 解決：重新設計問卷（反向題、不同量表格式）或增加其他方法

---

**準則 4：異特質相關模式在同方法與異方法三角區應相似**

📌 **回應問題**：構念關係一致性 - 構念間的理論關係不應受測量方法影響

**邏輯**：
- 若理論預期「T1 與 T2 正相關，T1 與 T3 無關」
- 這個模式應該在所有方法中都成立

**範例檢查**：

**同方法三角區（問卷）**：
- T1-T2 = 0.60
- T1-T3 = 0.15
- T2-T3 = 0.55

**異方法三角區（問卷 vs 實驗）**：
- T1-T2 = 0.58
- T1-T3 = 0.12
- T2-T3 = 0.50

判斷：兩個三角區的相關排序一致（T1-T2 > T2-T3 > T1-T3）✓

**違反範例**：
- 若問卷中 T1-T2 = 0.80，但異方法 T1-T2 = 0.20
- 問題：問卷法虛增 T1-T2 關係（可能題目語意重疊）
- 影響：理論檢驗結果受方法選擇影響，缺乏穩健性

### Q2：MTMM 要求「至少兩個特質、兩個方法」，但實務上很難做到多方法設計，有簡化替代方案嗎？

#### 完整 MTMM 的實務挑戰

**理想設計**：3 特質 × 3 方法 = 9 種測量組合

❌ **實務困難**：
1. **成本高**：需要設計問卷、實驗、訪談三種工具
2. **樣本需求大**：需要同一批受試者完成所有測量（增加疲勞效應）
3. **時間長**：資料蒐集可能需要數週到數月

---

#### 簡化替代方案 1：最小 MTMM（2×2 設計）

**設計**：2 特質 × 2 方法

**範例**：驗證「知覺有用性」vs「知覺易用性」
- 方法 1：問卷
- 方法 2：實驗任務評分

**矩陣簡化**：

| | M1-T1 | M1-T2 | M2-T1 | M2-T2 |
|---|---|---|---|---|
| M1-T1 | 1.00 | 0.45 | **0.72** | 0.30 |
| M1-T2 | 0.45 | 1.00 | 0.28 | **0.68** |
| M2-T1 | **0.72** | 0.28 | 1.00 | 0.42 |
| M2-T2 | 0.30 | **0.68** | 0.42 | 1.00 |

**可驗證內容**：
✓ 收斂效度：0.72, 0.68 都 > 0.7
✓ 區辨效度：0.72 > 0.45（準則 3）
✓ 基本效度證據足夠

**限制**：
- 無法檢驗準則 4（需要至少 3 個特質才能看模式）
- 統計檢定力較弱

---

#### 簡化替代方案 2：單一特質多方法（1×3 設計）

**適用情境**：開發新量表，重點驗證單一構念的收斂效度

**範例**：驗證「AI 診斷信任度」
- 方法 1：問卷
- 方法 2：情境實驗評分
- 方法 3：實際採納率（行為觀察）

**檢驗重點**：
- M1-M2 相關 = 0.75
- M1-M3 相關 = 0.68
- M2-M3 相關 = 0.72

**結論標準**：
- 若三個方法兩兩相關都 > 0.7 → 收斂效度良好 ✓
- 無法驗證區辨效度（需要至少兩個特質）

**應用時機**：量表發展初期，先確保測到目標構念

---

#### 簡化替代方案 3：時間分離法（Temporal Separation）

**設計**：用「時間點」替代「方法」

**範例**：
- T1 時間點：問卷測量「使用意圖」
- T2 時間點（3 個月後）：問卷測量「實際使用」

**邏輯**：
- 若 T1 使用意圖能預測 T2 實際使用 → 預測效度 ✓
- 時間分離降低共同方法偏差（不同時間點填答，情緒狀態不同）

**優點**：
- 不需設計多種測量工具
- 適用於縱貫性研究

**限制**：
- 無法完全排除方法偏差（都是問卷法）
- 需要追蹤樣本（流失率問題）

---

#### 簡化替代方案 4：確認性因素分析（CFA）+ MTMM

**現代整合做法**：用 CFA 模型化 MTMM 矩陣

**設計步驟**：
1. 收集 2 特質 × 2 方法資料
2. 建立 CFA 模型：
   - **特質因子**：T1（知覺有用性）、T2（知覺易用性）
   - **方法因子**：M1（問卷）、M2（實驗）
3. 檢驗模型配適度

**CFA-MTMM 模型**：

```
特質因子 T1 → M1-T1 題目 1, 2, 3
特質因子 T1 → M2-T1 實驗分數
方法因子 M1 → M1-T1 題目 1, 2, 3（控制問卷偏差）

比較模型：
- 模型 1（特質 + 方法）vs 模型 2（僅特質）
- 若模型 1 配適度顯著更好 → 存在方法偏差
```

**優點**：
- 量化特質變異 vs 方法變異比例
- 提供統計顯著性檢定
- 成為現代 MTMM 驗證主流

**軟體**：AMOS, Mplus, lavaan (R)

---

#### 實務建議：階段性效度驗證策略

| 研究階段 | 效度驗證重點 | 最小設計 | 推薦設計 |
|---------|------------|---------|---------|
| **量表開發** | 收斂效度 | 1 特質 × 2 方法 | 1 特質 × 3 方法 + CFA |
| **構念區辨** | 區辨效度 | 2 特質 × 2 方法 | 3 特質 × 2 方法 + CFA |
| **理論檢驗** | 關係穩健性 | 主要變數用 2 方法 | 全模型 CFA-MTMM |
| **跨文化驗證** | 測量恆等性 | 2 文化 × 1 方法 + 多群組 CFA | 2 文化 × 2 方法 MTMM |

**成本效益考量**：
- 博士論文：建議完整 MTMM（3×3）展現方法學嚴謹性
- 期刊論文：最少 2×2 + CFA 補充證據
- 實務研究：單一特質多方法（1×3）已足夠

### Q3：MTMM 判讀時，「效度對角線應該多高」才算合格？有量化標準嗎？

#### 經典標準：Campbell-Fiske (1959) 的模糊建議

原文表述：
> "Validity diagonal values should be significantly different from zero and sufficiently large"

❌ **問題**：
- 「sufficiently large」沒有明確數值
- 1959 年代缺乏大樣本元分析，無法給經驗基準

---

#### 現代經驗標準：後設分析歸納

**Cohen (1988) 效果量標準**（通用）：

| 效果量 | Pearson r | 解釋 | MTMM 效度對角線評價 |
|-------|-----------|------|-------------------|
| 小 | 0.10 - 0.30 | 微弱關係 | ✗ 收斂效度不足 |
| 中 | 0.30 - 0.50 | 中等關係 | △ 勉強可接受 |
| 大 | 0.50 - 0.70 | 強烈關係 | ✓ 良好收斂效度 |
| 很大 | &gt; 0.70 | 非常強關係 | ✓✓ 優秀收斂效度 |

---

**IS 領域專屬標準**：Straub et al. (2004) 整理

基於 IS 頂級期刊 100+ 篇 MTMM 研究：

| MTMM 相關類型 | 建議門檻 | 理想值 | 警示值 |
|--------------|---------|--------|--------|
| **同特質異方法**（效度對角線） | &gt; 0.50 | &gt; 0.70 | &lt; 0.30（拒絕） |
| **異特質同方法** | &lt; 0.70 | &lt; 0.50 | &gt; 0.80（方法偏差過大） |
| **異特質異方法** | &lt; 0.30 | &lt; 0.10 | &gt; 0.50（構念重疊） |

---

#### 情境調整標準：考量測量方法差異

**主觀 vs 主觀方法**（如問卷 vs 訪談）：
- 預期效度對角線：0.70 - 0.85
- 理由：都是自陳法，共享認知基礎

**主觀 vs 客觀方法**（如問卷 vs 系統記錄）：
- 預期效度對角線：0.50 - 0.70
- 理由：測量角度不同，相關自然較低

**客觀 vs 客觀方法**（如系統記錄 A vs 系統記錄 B）：
- 預期效度對角線：0.80 - 0.95
- 理由：都是客觀指標，應高度一致

---

#### 範例判讀：醫療 AI 信任度 MTMM

**情境**：驗證「AI 信任度」構念

| 方法組合 | 效度對角線 | 判讀 | 理由 |
|---------|-----------|------|------|
| 問卷 vs 情境實驗評分 | r = 0.68 | ✓ 合格 | 主觀 vs 主觀，0.68 &gt; 0.50 門檻 |
| 問卷 vs 實際採納率 | r = 0.45 | △ 勉強 | 主觀 vs 客觀，0.45 接近 0.50 門檻 |
| 情境實驗 vs 實際採納率 | r = 0.52 | ✓ 合格 | 主觀 vs 客觀，0.52 &gt; 0.50 門檻 |

**綜合判斷**：
- 三個效度對角線都 &gt; 0.45，收斂效度基本成立
- 但問卷 vs 實際行為略低，建議補充質性資料解釋差異
- 可能原因：問卷測「態度性信任」，行為測「行為性信任」，兩者有差異

---

#### 統計顯著性 vs 實務顯著性

⚠️ **常見錯誤**：只看 p 值，不看相關係數大小

**範例**：
- 大樣本（n = 500）：r = 0.35, p < 0.001
- 小樣本（n = 50）：r = 0.68, p = 0.08

**傳統做法**：
- 選擇 r = 0.35（因為 p < 0.05）✗

**MTMM 正確做法**：
- 優先看效果量（r = 0.68 vs 0.35）
- p 值只是輔助（確認非抽樣誤差）
- **結論**：r = 0.68 更有實務意義，即使 p 值未達顯著

**建議**：
- 報告效度對角線時，同時提供 r 值、95% 信賴區間、p 值
- 範例：r = 0.68, 95% CI [0.52, 0.79], p < 0.001

---

#### 進階判讀：效度對角線的信賴區間

**問題**：r = 0.65 看起來不錯,但信賴區間很寬 [0.30, 0.85]，該接受嗎？

**判斷邏輯**：
- 95% CI 下界 = 0.30（低於 0.50 門檻）→ 效度證據不穩定
- **結論**：即使點估計 0.65 合格，樣本量不足導致不確定性高

**解決方案**：
1. 增加樣本量，縮小 CI
2. 設定最低可接受界限（SESOI）
   - 範例：CI 下界必須 > 0.40 才接受

**統計檢定**：
- H0: ρ ≤ 0.50（效度不足）
- H1: ρ > 0.50（效度充分）
- 若 95% CI 完全在 0.50 以上 → 拒絕 H0

---

#### 領域特定標準：醫療 AI vs 一般 IS

**醫療 AI 領域的特殊考量**：

| 情境 | 標準調整 | 理由 |
|------|---------|------|
| **診斷輔助系統** | 效度對角線需 &gt; 0.80 | 涉及病患安全，要求更高效度證據 |
| **行政管理系統** | 效度對角線 &gt; 0.60 即可 | 風險較低，容錯空間較大 |
| **跨專業團隊** | 異特質同方法應 &lt; 0.40 | 醫師 vs 護理師認知差異大，區辨效度更重要 |

**實例**：
- AI 診斷信任度（診斷輔助）：問卷 vs 實驗 r = 0.72
  - 一般 IS 標準：✓ 合格
  - 醫療 AI 標準：△ 建議提升至 0.80+

---

#### 實務建議：效度對角線判讀檢查表

**步驟 1：基本門檻檢查**
- [ ] 所有效度對角線 > 0.50？
- [ ] 至少 2/3 的效度對角線 > 0.70？

**步驟 2：方法差異調整**
- [ ] 主觀-主觀組合 > 0.70？
- [ ] 主觀-客觀組合 > 0.50？
- [ ] 客觀-客觀組合 > 0.80？

**步驟 3：統計穩健性檢查**
- [ ] 效度對角線 p < 0.05？
- [ ] 95% CI 下界 > 0.40？
- [ ] 樣本量 > 100（確保檢定力）？

**步驟 4：領域標準檢查**
- [ ] 符合目標期刊的效度要求？
- [ ] 高風險應用情境（如醫療）是否達更高標準？

**決策樹**：
- 4 個步驟都通過 → ✓✓ 優秀效度
- 步驟 1-3 通過 → ✓ 可接受效度
- 步驟 1 未通過 → ✗ 效度不足，需重新設計測量

## 🤔 第四張：例外卡

### 例外 1：MTMM 預設「特質與方法獨立」，但實務中兩者常交互作用

#### 理論假設 vs 現實複雜性

**MTMM 理想模型**：
```
觀察分數 = 特質效應 + 方法效應 + 誤差
（假設特質與方法相加，無交互作用）
```

❌ **現實情境**：特質 × 方法交互作用普遍存在

**範例 1：知覺有用性的方法敏感性**

| 受試者類型 | 問卷評分（M1） | 實驗任務評分（M2） | 效度對角線 |
|-----------|--------------|----------------|----------|
| **高科技素養** | 8 分 | 9 分 | r = 0.85 ✓ |
| **低科技素養** | 6 分 | 3 分 | r = 0.35 ✗ |

**交互作用解讀**：
- 高科技素養者：問卷與實驗評分一致（能準確自我報告）
- 低科技素養者：問卷高估（不知道自己不知道），實驗才暴露真實能力
- **結論**：特質（有用性知覺）的測量受方法（問卷 vs 實驗）與個人特性（科技素養）交互影響

**MTMM 盲點**：
- 矩陣相關係數無法偵測這種異質性
- 可能平均後得到「中等收斂效度 r = 0.60」，掩蓋高低素養組的巨大差異

---

**範例 2：文化情境下的方法偏好**

**情境**：跨文化驗證「團隊合作意願」

| 文化背景 | 問卷自陳（M1） | 同儕評分（M2） | 效度對角線 |
|---------|--------------|--------------|----------|
| **集體主義文化** | 謙虛低估（6 分） | 同儕肯定（8 分） | r = 0.40 ✗ |
| **個人主義文化** | 自信高估（8 分） | 同儕中立（7 分） | r = 0.75 ✓ |

**交互作用機制**：
- 集體主義文化：自陳法受「謙虛規範」影響，系統性低估
- 個人主義文化:自陳法鼓勵「自我肯定」,與他人評價一致
- **MTMM 困境**：同一測量工具在不同文化效度不同

---

#### 解決方案：階層 MTMM 或潛在調節模型

**進階方法 1：多群組 MTMM**

**步驟**：
1. 將樣本分組（高 vs 低科技素養）
2. 各組分別計算 MTMM 矩陣
3. 比較兩組的效度對角線

**發現範例**：
- 高素養組：所有效度對角線 > 0.75
- 低素養組：所有效度對角線 < 0.45
- **結論**：測量工具僅對高素養組有效

---

**進階方法 2：CFA-MTMM + 潛在交互作用**

**模型設定**：
```
觀察分數 = β1(特質) + β2(方法) + β3(特質 × 方法) + ε
```

**檢驗步驟**：
1. 模型 1（無交互作用）vs 模型 2（有交互作用）
2. 若模型 2 配適度顯著更好 → 存在交互作用
3. 檢視 β3 係數：哪些特質受哪些方法影響

**軟體**：Mplus (LMS method), lavaan (2-stage approach)

---

### 例外 2：「方法」定義模糊 - 什麼算不同方法？

#### 方法差異的連續體困境

**問題**：MTMM 要求「不同方法」，但何謂「不同」？

**光譜範例**：

| 測量工具對比 | 是否為「不同方法」？ | Campbell-Fiske 立場 | 現代爭議 |
|------------|------------------|------------------|---------|
| 紙本問卷 vs 線上問卷 | ✗ 同一方法 | 介質不影響本質 | 數位落差可能影響（年長者不熟線上） |
| Likert 5 點 vs 7 點量表 | ✗ 同一方法 | 都是自陳法 | 尺度敏感度不同可能影響 |
| 自評 vs 他評問卷 | ✓ 不同方法 | 資訊來源不同 | ✓ 廣泛接受 |
| 問卷 vs 訪談 | ✓ 不同方法 | 資料蒐集方式不同 | ✓ 廣泛接受 |
| 問卷 vs 實驗 | ✓ 不同方法 | 情境控制差異大 | ✓ 廣泛接受 |
| 態度問卷 vs 行為觀察 | ✓✓ 非常不同 | 主觀 vs 客觀 | ✓ 最強區辨 |

---

#### 爭議案例：時間點算不同方法嗎？

**情境**：用「T1 時間問卷」vs「T2 時間問卷」驗證收斂效度

**支持觀點**：
✓ 時間分離降低共同方法偏差（情緒狀態改變）
✓ 若構念穩定，T1 vs T2 應高度相關

**反對觀點**：
✗ 本質上還是同一問卷,只是重測信度
✗ 若構念本身隨時間變化（如「當下心情」），低相關不代表效度差

**IS 領域共識**（Podsakoff et al. 2012）：
- **程序性分離（Procedural Remedies）**：時間、來源、情境分離
- **不完全等同於「不同方法」**，但可作為補充證據
- 最好搭配真正的方法差異（如問卷 + 實驗）

---

#### 實務建議：方法差異性檢查表

**最小方法差異標準**（至少滿足 2 項）：

- [ ] **資料來源不同**：自評 vs 他評、員工 vs 主管
- [ ] **蒐集方式不同**：問卷 vs 訪談 vs 觀察
- [ ] **情境控制不同**：自然情境 vs 實驗操弄
- [ ] **主客觀性質不同**：主觀知覺 vs 客觀記錄
- [ ] **認知負荷不同**：簡單勾選 vs 複雜任務表現

**強方法差異範例**（IS 領域最佳實務）：
1. 問卷（自評知覺）+ 系統日誌（客觀使用）
2. 情境實驗（控制情境）+ 田野觀察（自然情境）
3. 量化問卷 + 質性訪談編碼

### 例外 3：MTMM 無法處理「形成性構念」（Formative Constructs）

#### 反映性 vs 形成性構念的測量邏輯差異

**反映性構念（Reflective）**：指標是構念的反映結果

```
        構念（知覺有用性）
          ↙   ↓   ↘
      指標1 指標2 指標3
    （提升效率）（改善品質）（節省時間）
```

- **特性**：指標間應高度相關（內部一致性）
- **MTMM 適用**：✓ 可用不同方法測量同一構念

---

**形成性構念（Formative）**：指標共同形成構念

```
      指標1 指標2 指標3
    （硬體）（軟體）（網路）
          ↘   ↓   ↙
        構念（系統品質）
```

- **特性**：指標間可以不相關（涵蓋不同面向）
- **MTMM 問題**：✗ 不同方法可能測到不同面向組合

---

#### 形成性構念的 MTMM 困境

**範例**：「系統品質」形成性構念

**問卷測量**（M1）：
- 硬體速度（8 分）
- 軟體穩定性（6 分）
- 網路連線（4 分）
- → 綜合評分 = 6 分

**專家稽核**（M2）：
- 硬體速度（9 分，客觀測試）
- 軟體穩定性（5 分，錯誤率）
- 網路連線（7 分，頻寬測試）
- → 綜合評分 = 7 分

**效度對角線計算**：
- 問卷總分 vs 專家總分：r = 0.45（偏低）

❌ **錯誤結論**：收斂效度不足

✓ **正確解讀**：
- 問卷與專家對「硬體」一致（r = 0.85）
- 但對「網路」評價不同（問卷受主觀感受影響）
- **形成性構念**：不同方法可能賦予不同指標權重，總分自然不同

---

#### 解決方案：指標層次 MTMM

**傳統 MTMM**：比較構念層次總分
**形成性 MTMM**：比較各指標層次分數

**修正矩陣**：

| | M1-硬體 | M1-軟體 | M1-網路 | M2-硬體 | M2-軟體 | M2-網路 |
|---|---|---|---|---|---|---|
| M1-硬體 | 1.00 | 0.15 | 0.20 | **0.85** | 0.18 | 0.12 |
| M1-軟體 | 0.15 | 1.00 | 0.10 | 0.20 | **0.78** | 0.15 |
| M1-網路 | 0.20 | 0.10 | 1.00 | 0.25 | 0.18 | **0.55** |

**解讀**：
- 硬體、軟體指標收斂效度良好（0.85, 0.78）
- 網路指標收斂效度較低（0.55）→ 主觀知覺與客觀測量差異大
- **結論**：「系統品質」構念部分有效，網路指標需改進

---

#### IS 領域形成性構念範例

| 構念 | 形成性指標 | MTMM 挑戰 |
|------|-----------|----------|
| **系統品質** | 硬體 + 軟體 + 網路 | 不同方法可能側重不同面向 |
| **資訊品質** | 準確性 + 即時性 + 完整性 | 專家 vs 使用者對「準確」定義不同 |
| **組織支持** | 高層支持 + 資源投入 + 政策配合 | 問卷 vs 文件分析測到不同證據 |

**建議**：
- 形成性構念不宜用總分做 MTMM
- 應檢驗各指標的收斂效度
- 或改用「效度網絡」（Nomological Network）驗證

### 例外 4：MTMM 對「動態構念」無能為力

#### 靜態效度假設 vs 構念時間演化

**MTMM 靜態假設**：
- 構念在測量期間穩定不變
- T1 時間的「信任」= T2 時間的「信任」

❌ **動態構念現實**：
- 初期信任 ≠ 持續使用後的信任
- 測量方法可能對不同時期構念敏感度不同

---

#### 範例：AI 診斷系統的「信任演化」

**階段 1（初期接觸，1 週）**：
- 問卷測信任 = 7 分（基於品牌聲譽的認知信任）
- 實際採納率 = 30%（觀望態度）
- 效度對角線：r = 0.40（收斂效度低）

**階段 2（適應期，1 個月）**：
- 問卷測信任 = 6 分（發現系統限制，認知調整）
- 實際採納率 = 60%（習慣養成，行為信任建立）
- 效度對角線：r = 0.32（更低！）

**階段 3（穩定期，3 個月）**：
- 問卷測信任 = 8 分（認知與行為一致）
- 實際採納率 = 75%
- 效度對角線：r = 0.78（收斂效度回升）

**MTMM 盲點**：
- 若只在「階段 2」測量 → 錯誤結論「效度不足」
- 實際：構念本身在演化，認知信任 vs 行為信任的關係隨時間改變

---

#### 解決方案：縱貫 MTMM（Longitudinal MTMM）

**設計**：多時間點 × 多特質 × 多方法

**簡化範例**：2 時間點 × 2 特質 × 2 方法

| | T1-M1 | T1-M2 | T2-M1 | T2-M2 |
|---|---|---|---|---|
| T1-M1（初期問卷信任） | 1.00 | 0.40 | 0.55 | 0.35 |
| T1-M2（初期行為信任） | 0.40 | 1.00 | 0.30 | 0.60 |
| T2-M1（後期問卷信任） | 0.55 | 0.30 | 1.00 | 0.78 |
| T2-M2（後期行為信任） | 0.35 | 0.60 | 0.78 | 1.00 |

**洞察**：
1. **初期效度對角線**：T1-M1 vs T1-M2 = 0.40（低）
2. **後期效度對角線**：T2-M1 vs T2-M2 = 0.78（高）
3. **跨時間追蹤**：T1-M2 vs T2-M2 = 0.60（行為信任穩定性）
4. **結論**：信任構念在初期與後期有不同測量特性

## 📊 本研究的研究方法

### 方法類型

**方法學論文 + 概念整合**

**資料來源**：
- **理論回顧**：心理計量學效度理論發展脈絡
- **方法學整合**：收斂效度、區辨效度、多元操作主義哲學基礎
- **實例推導**：以假設性構念設計 MTMM 矩陣範例

---

### 核心方法論：多特質多方法矩陣（MTMM）

#### 設計邏輯

**交叉設計（Crossed Design）**：

```
        方法 1    方法 2    方法 3
特質 A   測量 A1   測量 A2   測量 A3
特質 B   測量 B1   測量 B2   測量 B3
特質 C   測量 C1   測量 C2   測量 C3
```

**完整矩陣結構**（3 特質 × 3 方法 = 9 種測量）：

| | M1-A | M1-B | M1-C | M2-A | M2-B | M2-C | M3-A | M3-B | M3-C |
|---|---|---|---|---|---|---|---|---|---|
| M1-A | **(R)** | HM | HM | **V** | HT | HT | **V** | HT | HT |
| M1-B | HM | **(R)** | HM | HT | **V** | HT | HT | **V** | HT |
| M1-C | HM | HM | **(R)** | HT | HT | **V** | HT | HT | **V** |
| M2-A | **V** | HT | HT | **(R)** | HM | HM | **V** | HT | HT |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

**符號說明**：
- **(R)**：信度係數（reliability diagonal）- 同一測量重測相關
- **V**：效度係數（validity diagonal）- 同特質異方法相關
- HM：異特質同方法相關（heterotrait-monomethod）
- HT：異特質異方法相關（heterotrait-heteromethod）

---

### 判讀準則的邏輯推導

**準則 1（收斂效度）**：

`r_AiAj > 0 (i ≠ j)`

其中 $A_i$ 和 $A_j$ 是特質 A 的不同方法測量

**意義**：同一構念用不同方法測量應顯著相關

---

**準則 2（區辨效度 - 異方法比較）**：

`r_AiAj > r_AiBj` (validity diagonal > heterotrait-heteromethod)

**意義**：同構念異方法相關應大於異構念異方法相關

---

**準則 3（區辨效度 - 同方法比較）**：

`r_AiAj > r_AiBi` (validity diagonal > heterotrait-monomethod)

**意義**：同構念異方法相關應大於異構念同方法相關（控制方法偏差）

---

**準則 4（模式一致性）**：

`Rank(r_AB, r_AC, r_BC)_HM ≈ Rank(r_AB, r_AC, r_BC)_HT`

**意義**：構念間相關的排序模式在同方法與異方法三角區應相似

---

### 實例研究設計（Campbell-Fiske 原文範例簡化）

**研究目標**：驗證心理特質測量效度

**特質（Traits）**：
- T1：外向性（Extraversion）
- T2：親和性（Agreeableness）

**方法（Methods）**：
- M1：自陳式問卷
- M2：同儕評分
- M3：專家觀察

**樣本**：100 位大學生（每位完成 3 種測量）

**結果矩陣（假設數據）**：

| | M1-T1 | M1-T2 | M2-T1 | M2-T2 | M3-T1 | M3-T2 |
|---|---|---|---|---|---|---|
| M1-T1 | (.85) | .42 | .68 | .30 | .65 | .28 |
| M1-T2 | .42 | (.88) | .35 | .72 | .32 | .70 |
| M2-T1 | .68 | .35 | (.80) | .38 | .75 | .25 |
| M2-T2 | .30 | .72 | .38 | (.82) | .28 | .78 |
| M3-T1 | .65 | .32 | .75 | .28 | (.78) | .35 |
| M3-T2 | .28 | .70 | .25 | .78 | .35 | (.80) |

**判讀結果**：
- ✓ 準則 1：效度對角線（.68, .72, .75, .78）都 > 0.5
- ✓ 準則 2：效度對角線 > 異特質異方法（如 .68 > .30）
- ✓ 準則 3：效度對角線 > 異特質同方法（如 .68 > .42）
- ✓ 準則 4：T1-T2 相關在三個方法中排序一致（.42, .38, .35）

**結論**：外向性與親和性具備良好的收斂與區辨效度

---

### 統計分析技術

**1. 相關係數矩陣**：
- 使用 Pearson r（連續變數）或 Spearman ρ（序位變數）
- 檢定 H0: ρ = 0（雙尾檢定）

**2. 效果量解釋**（Cohen 1988）：
- 小效果：r = .10 - .30
- 中效果：r = .30 - .50
- 大效果：r ≥ .50

**3. 信賴區間**（Fisher's Z 轉換）：

`Z = 0.5 × ln((1+r)/(1-r))`, `SE_Z = 1/√(n-3)`

**4. 矩陣模式檢驗**（視覺 + 量化）：
- 視覺掃描：檢查對角線數值是否最大
- 量化：計算效度對角線平均值 vs 其他三角區平均值

---

### 方法學限制

1. **樣本需求**：
   - 3 特質 × 3 方法 = 9 種測量
   - 每位受試者需完成所有測量（疲勞效應、練習效應）

2. **統計檢定力**：
   - MTMM 依賴多次相關比較
   - 無整體顯著性檢定（需目視判斷）

3. **方法獨立性假設**：
   - 假設特質與方法效應可分離（實務常違反）

---

### 現代延伸：CFA-MTMM

**結構方程模型化**：

**模型設定**：
- **特質因子**：F1（外向性）、F2（親和性）
- **方法因子**：M1（問卷）、M2（同儕）、M3（專家）

**測量模型**：

`X_ij = λ_Ti × F_T + λ_Mi × F_M + ε_ij`

其中：
- `X_ij`：第 i 個特質用第 j 個方法的測量
- `F_T`：特質因子
- `F_M`：方法因子
- `λ`：因素負荷量

**模型比較**：
- M1（特質 + 方法）vs M0（僅特質）
- 若 χ² 差異顯著 → 方法變異存在

**優點**：
- 量化特質變異 vs 方法變異比例
- 提供整體配適度指標（CFI, RMSEA）
- 統計顯著性檢定

**軟體**：LISREL, AMOS, Mplus, lavaan (R)

- 報告方法變異比例

✓ **提升**：效度證據從「單一來源」→「多元三角驗證」

### 2. 共同方法偏差（CMV）覺醒

**Podsakoff et al. (2003) 經典回顧**受 MTMM 啟發：

**CMV 嚴重性證據**：
- 分析 IS 領域 216 篇論文
- 平均「異特質同方法」相關 = 0.52（過高！）
- **推論**：約 27% 的變異來自方法偏差，非構念關係

**MTMM 診斷 CMV 的機制**：

| MTMM 矩陣模式 | CMV 嚴重程度 | 建議 |
|--------------|------------|------|
| 異特質同方法 &lt; 0.30 | 輕微 | 無需額外處理 |
| 0.30 - 0.50 | 中等 | 程序補救（分時測量） |
| 0.50 - 0.70 | 嚴重 | 統計補救（控制方法因子） |
| &gt; 0.70 | 極嚴重 | 重新設計研究 |

---

#### IS 研究的 CMV 補救策略（基於 MTMM 原則）

**程序性補救（Procedural Remedies）**：

1. **來源分離**：
   - 自變數：員工自評
   - 依變數：主管評分
   - MTMM 邏輯：不同資料來源 = 不同方法

2. **時間分離**：
   - T1：測量知覺有用性
   - T2（2 週後）：測量使用意圖
   - MTMM 邏輯：時間間隔降低情緒狀態連動

3. **心理分離**：
   - 匿名保證
   - 反向題混合
   - MTMM 邏輯：降低一致性動機（tendency to agree）

---

**統計補救（Statistical Remedies）**：

1. **Harman 單因子檢定**：
   - 所有題目做探索性因素分析
   - 若單一因子解釋變異 > 50% → 嚴重 CMV

2. **潛在方法因子控制（CFA-MTMM 應用）**：
   ```
   測量模型：
   X = λ(特質因子) + γ(方法因子) + ε

   比較：
   - 模型 1（無方法因子）
   - 模型 2（加入方法因子）
   - 若 R² 顯著提升 → CMV 存在
   ```

3. **Marker Variable 技術**：
   - 加入理論上不相關的變數（如「喜歡的顏色」）
   - 若該變數與研究變數相關 > 0.30 → 方法偏差

---

### 3. IS 核心構念的效度重新檢驗

#### TAM（科技接受模型）的 MTMM 驗證歷程

**Davis (1989) 原始 TAM**：
- 僅用問卷測量「知覺有用性」、「知覺易用性」
- 效度證據：因素分析（兩構念分開）

⚠️ **質疑**：
- 兩構念問卷相關常 > 0.70
- 可能是語意重疊 or 真實構念關係？

---

**Straub et al. (1995) MTMM 驗證**：

**設計**：
- 特質：知覺有用性、知覺易用性
- 方法：問卷、實驗任務評分、實際使用時間

**結果矩陣（簡化）**：

| | M1-有用性 | M1-易用性 | M2-有用性 | M2-易用性 |
|---|---|---|---|---|
| M1-有用性 | 1.00 | 0.65 | **0.58** | 0.28 |
| M1-易用性 | 0.65 | 1.00 | 0.30 | **0.62** |
| M2-有用性 | **0.58** | 0.30 | 1.00 | 0.38 |
| M2-易用性 | 0.28 | **0.62** | 0.38 | 1.00 |

**發現**：
- ✓ 效度對角線（0.58, 0.62）> 0.5（收斂效度成立）
- ⚠️ 異特質同方法（0.65）> 效度對角線（0.58）
  - 違反準則 3 → 問卷法存在方法偏差
- ✓ 異方法比較：0.58 > 0.28（準則 2 成立）

**結論**：
- TAM 兩構念確實獨立存在（區辨效度）
- 但問卷法會虛增兩構念相關
- **建議**：TAM 研究應加入行為測量

---

**後續影響**：
- Venkatesh et al. (2003) UTAUT：整合 8 個模型時，特別強調「行為意圖 → 實際使用」需用客觀紀錄驗證
- 成為 IS 行為研究的黃金標準

### 4. 跨文化 IS 研究的測量恆等性檢驗

**問題**：美國開發的 TAM 量表是否適用於亞洲？

**MTMM 擴展應用**：文化視為「方法」

**McCoy et al. (2005) 跨文化 MTMM**：

**設計**：
- 特質：知覺有用性、知覺易用性
- 「方法」：美國樣本、台灣樣本、日本樣本

**檢驗邏輯**：
- 若同構念跨文化相關高 → 構念意義穩定
- 若跨文化相關低 → 文化改變構念詮釋

**結果**：
- 知覺有用性（美 vs 台）= 0.72 ✓
- 知覺易用性（美 vs 日）= 0.48 ⚠️
  - 發現：日本文化對「易用性」定義包含「社會和諧」，非僅個人便利

**影響**：
- 推動「文化調適量表」發展
- 要求跨文化研究報告測量恆等性（Measurement Invariance）

### 給 IS 研究者的 MTMM 實施指南

#### 步驟 1：確定研究目標

**效度驗證類型**：

| 目標 | 最小設計 | 推薦設計 | 時程 |
|------|---------|---------|------|
| 新量表開發 | 1 特質 × 2 方法 | 1 特質 × 3 方法 + CFA | 6-12 個月 |
| 構念區辨驗證 | 2 特質 × 2 方法 | 3 特質 × 2 方法 + CFA | 3-6 個月 |
| CMV 診斷 | 主要變數 × 2 方法 | 全模型 CFA-MTMM | 追加 3 個月 |
| 跨文化適用性 | 2 文化 × 1 方法 + 多群組 CFA | 2 文化 × 2 方法 MTMM | 12-18 個月 |

---

#### 步驟 2：選擇適當的「方法」組合

**高效方法配對**（基於 IS 文獻後設分析）：

**配對 1：問卷 + 實驗**
- 優點：主觀知覺 vs 控制情境，方法差異大
- 缺點：實驗情境可能不真實
- 適用：TAM, TPB 等態度-行為模型

**配對 2：自評 + 他評**
- 優點：相同構念不同視角
- 缺點：他評者可觀察性限制
- 適用：團隊協作、知識分享研究

**配對 3：問卷 + 系統日誌**
- 優點：主觀 vs 客觀，CMV 最小
- 缺點：日誌可能缺乏心理狀態資訊
- 適用：系統使用行為研究

**配對 4：當下測量 + 經驗取樣法（ESM）**
- 優點：減少回憶偏差
- 缺點：受試者負擔大
- 適用：情境動態研究

---

#### 步驟 3：資料蒐集設計

**樣本量規劃**：

**MTMM 最小樣本量公式**（Westland 2010）：

`n ≥ (50 × (p + q)) / t`

其中：
- p = 特質數
- q = 方法數
- t = 每個測量的題目數

**範例**：
- 3 特質 × 2 方法，每個 5 題
- `n ≥ (50 × (3 + 2)) / 5 = 50`

**推薦樣本量**（考量檢定力）：

| 設計複雜度 | 最小樣本 | 推薦樣本 | 理想樣本 |
|-----------|---------|---------|---------|
| 2×2 | 100 | 200 | 300+ |
| 3×2 | 150 | 250 | 400+ |
| 3×3 | 200 | 300 | 500+ |

---

**測量順序設計**（減少順序效應）：

**拉丁方格設計（Latin Square）**：

| 受試者組別 | 第一個方法 | 第二個方法 | 第三個方法 |
|-----------|----------|----------|----------|
| 組 1 | 問卷 | 實驗 | 訪談 |
| 組 2 | 實驗 | 訪談 | 問卷 |
| 組 3 | 訪談 | 問卷 | 實驗 |

**時間間隔**：
- 主觀測量間隔：至少 1 週（降低記憶效應）
- 主觀 vs 客觀：可同時進行
- 縱貫設計：至少 1 個月間隔

---

#### 步驟 4：資料分析

**工具選擇**：

| 分析需求 | R 套件 | SPSS | SEM 軟體 |
|---------|-------|------|---------|
| 相關矩陣 | psych::corr.test() | Correlate | - |
| MTMM 矩陣視覺化 | corrplot | - | - |
| 效度對角線檢定 | cocor::cocor() | - | - |
| CFA-MTMM | lavaan | AMOS | Mplus |
| CMV 診斷 | semTools::measurementInvariance() | - | Mplus |

---

**R 語言範例程式碼**：

```r
# 載入套件
library(psych)
library(lavaan)
library(corrplot)

# 假設資料：3 特質 × 2 方法
# M1_T1 = 方法 1 測量特質 1
# M1_T2 = 方法 1 測量特質 2
# ...

# 1. 計算相關矩陣
mtmm_corr <- cor(data[, c("M1_T1", "M1_T2", "M1_T3",
                           "M2_T1", "M2_T2", "M2_T3")])

# 2. 視覺化 MTMM 矩陣
corrplot(mtmm_corr, method = "number", type = "full",
         tl.col = "black", tl.srt = 45,
         addCoef.col = "black", number.cex = 0.7)

# 3. 效度對角線檢定
library(cocor)
# 檢定：同特質異方法 vs 異特質同方法
cocor(~M1_T1 + M2_T1 | M1_T1 + M1_T2, data = data)

# 4. CFA-MTMM 模型
mtmm_model <- '
  # 特質因子
  Trait1 =~ M1_T1 + M2_T1
  Trait2 =~ M1_T2 + M2_T2
  Trait3 =~ M1_T3 + M2_T3

  # 方法因子
  Method1 =~ M1_T1 + M1_T2 + M1_T3
  Method2 =~ M2_T1 + M2_T2 + M2_T3

  # 因子間相關
  Trait1 ~~ Trait2 + Trait3
  Method1 ~~ Method2
'

fit <- cfa(mtmm_model, data = data)
summary(fit, fit.measures = TRUE, standardized = TRUE)

# 5. 計算特質 vs 方法變異比例
# 提取標準化因素負荷量
std_loadings <- standardizedSolution(fit)

# 特質變異 = (特質負荷量)^2
# 方法變異 = (方法負荷量)^2
```

---

#### 步驟 5：結果報告

**學術論文必備要素**：

**方法部分**：
1. MTMM 設計理由：
   > "為驗證 X 構念的收斂與區辨效度，本研究採用 Campbell-Fiske (1959) 多特質多方法矩陣設計..."

2. 方法選擇說明：
   > "選擇問卷（主觀知覺）與系統日誌（客觀行為）作為兩種方法，以最大化方法差異，降低共同方法偏差..."

3. 樣本與程序：
   > "200 位參與者完成兩種測量，時間間隔 2 週，採拉丁方格設計平衡順序效應..."

---

**結果部分**：

**表格：MTMM 相關矩陣**

| | M1-T1 | M1-T2 | M2-T1 | M2-T2 |
|---|---|---|---|---|
| M1-T1 | (.88) | .42 | **.68*** | .30 |
| M1-T2 | .42 | (.85) | .28 | **.72*** |
| M2-T1 | **.68*** | .28 | (.80) | .38 |
| M2-T2 | .30 | **.72*** | .38 | (.82) |

*註：對角線為信度係數，粗體為效度對角線，*** p < .001*

**文字說明**：
> "效度對角線（r = .68 - .72）顯著高於異特質異方法相關（r = .28 - .38），支持收斂效度（Campbell-Fiske 準則 2）。效度對角線亦高於異特質同方法相關（r = .42），符合區辨效度標準（準則 3）。"

---

**CFA-MTMM 補充證據**：

**表格：模型配適度比較**

| 模型 | χ² | df | CFI | RMSEA | SRMR | Δχ² |
|------|----|----|-----|-------|------|-----|
| M1（僅特質因子） | 185.2 | 45 | .88 | .095 | .078 | - |
| M2（特質 + 方法因子） | 92.3 | 40 | .96 | .058 | .042 | 92.9*** |

*註：Δχ² 顯著表示加入方法因子顯著改善配適度*

**文字說明**：
> "CFA-MTMM 分析顯示，加入方法因子後模型配適度顯著提升（Δχ² = 92.9, p < .001），表示測量資料包含方法變異。特質因子平均解釋變異為 68%，方法因子為 22%，顯示特質訊號強於方法雜訊。"

---

#### 步驟 6：效度不足時的補救措施

**情境 1：收斂效度不足**（效度對角線 < 0.5）

**可能原因**：
1. 兩種方法測量不同構念面向
2. 測量工具信度過低
3. 樣本異質性高

**補救方案**：
- [ ] 重新檢視構念定義，確保兩種方法對準同一面向
- [ ] 提升測量信度（增加題目、改進題目品質）
- [ ] 分群分析（檢查不同子群效度是否不同）
- [ ] 質性訪談確認受試者對構念的理解

---

**情境 2：區辨效度不足**（異特質同方法相關 > 效度對角線）

**可能原因**：
1. 方法偏差過大
2. 構念定義重疊
3. 題目語意相似

**補救方案**：
- [ ] 程序補救：分時測量、不同資料來源
- [ ] 題目重新設計：降低語意重疊
- [ ] 統計補救：控制潛在方法因子
- [ ] 理論重新檢視：兩構念是否真為獨立？

---

**情境 3：方法變異過大**（CFA-MTMM 方法因子變異 > 50%）

**補救方案**：
1. **短期**：報告時控制方法因子，檢驗構念關係是否仍成立
   ```r
   # 控制方法因子後的構念關係
   structural_model <- '
     Outcome ~ Trait1 + Trait2 + Method1 + Method2
   '
   ```

2. **長期**：發展方法偏差更小的測量工具
   - 範例：改用隱式聯結測驗（IAT）取代自陳問卷

---

### 醫療 AI 領域的 MTMM 應用範例

#### 情境：驗證「AI 診斷信任度量表」

**研究背景**：
- 開發新量表測量醫師對 AI 診斷系統的信任
- 需要驗證構念效度

**MTMM 設計**：

**特質（構念）**：
- T1：能力信任（competence trust）- 相信 AI 有能力做出正確診斷
- T2：善意信任（benevolence trust）- 相信 AI 設計符合醫師利益

**方法**：
- M1：自陳式問卷（7 點 Likert 量表）
- M2：情境模擬實驗（呈現 10 個病例，評分信任程度）
- M3：實際採納率（3 個月追蹤，AI 建議採納比例）

---

**資料蒐集**：
- 樣本：250 位醫師
- 程序：
  1. T0：完成問卷（M1）
  2. T0 + 1 週：完成情境實驗（M2）
  3. T0 + 3 個月：蒐集實際使用紀錄（M3）

---

**結果矩陣**：

| | M1-能力 | M1-善意 | M2-能力 | M2-善意 | M3-能力 | M3-善意 |
|---|---|---|---|---|---|---|
| M1-能力 | (.92) | .38 | **.75** | .25 | **.68** | .20 |
| M1-善意 | .38 | (.89) | .30 | **.70** | .22 | **.58** |
| M2-能力 | **.75** | .30 | (.85) | .35 | **.72** | .28 |
| M2-善意 | .25 | **.70** | .35 | (.82) | .25 | **.65** |
| M3-能力 | **.68** | .22 | **.72** | .25 | (.78)ᵃ | .30 |
| M3-善意 | .20 | **.58** | .28 | **.65** | .30 | (.70)ᵃ |

*註：ᵃ M3 為客觀行為，信度以 3 個月內採納率穩定性計算*

---

**效度判讀**：

**收斂效度**（準則 1）：
- 能力信任：M1-M2 = .75 ✓, M1-M3 = .68 ✓, M2-M3 = .72 ✓
- 善意信任：M1-M2 = .70 ✓, M1-M3 = .58 △, M2-M3 = .65 ✓
- **結論**：問卷與實驗高度一致，但與實際行為相關略低（主觀 vs 客觀差異）

**區辨效度**（準則 2 & 3）：
- 異特質同方法（M1）：能力 vs 善意 = .38 ✓（低於效度對角線 .68-.75）
- 異特質異方法：.20-.30 ✓（更低）
- **結論**：能力信任與善意信任具備區辨效度

---

**CFA-MTMM 補充分析**：

```r
mtmm_model <- '
  # 特質因子
  Competence =~ M1_comp + M2_comp + M3_comp
  Benevolence =~ M1_bene + M2_bene + M3_bene

  # 方法因子
  Survey =~ M1_comp + M1_bene
  Experiment =~ M2_comp + M2_bene
  Behavior =~ M3_comp + M3_bene
'
```

**結果**：
- 特質變異：能力信任 72%, 善意信任 68%
- 方法變異：問卷 18%, 實驗 15%, 行為 25%
- **解讀**：特質訊號強於方法雜訊，但行為測量有獨特變異（可能受臨床情境影響）

---

**實務啟示**：

1. **量表有效**：問卷工具可有效測量兩種信任
2. **主觀-客觀落差**：問卷高估實際信任（善意信任尤其明顯）
   - 改進：加入校準機制（顯示醫師實際採納率，促進自我覺察）
3. **方法建議**：
   - 學術研究：建議使用問卷 + 實驗（收斂效度最高）
   - 實務評估：直接用行為指標（避免社會期許偏差）
